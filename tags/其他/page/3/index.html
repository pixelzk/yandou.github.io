据说说有雪茄的味道，我倒是没有抽出雪茄味。Unscented版本的DF说是原味（Unscented版本），实际上也有湖区味。但这个湖区味很特别很神秘，和其他GH草不同。开火之初，似印度的焚香味，第二三口立刻香气减弱变成很宜人的了，但不喜欢湖区味的依然不喜欢。很香甜醇厚的一款，味道到最后都很稳定。第一次抽新开的，我差点被击倒，反正晕了。说明虽然醇厚，但焦油尼古丁力量强大。后来保湿醒草一二个月，越来越美味。窖藏，不要让它起白霜，密封和保持湿度，让时间将它变得更加乌黑油亮，味道比出霜的更加醇厚甘甜丰富。烟草这东西各人喜欢不同，我喜欢的不一定你喜欢。

打开TR，把成千上万的烟草按照力道降序排列，GH在前50中只怕是绝对优势，而GH每个草又都有调味和不调味两种，像这个不调味的，只要没被放倒，嘴巴里是非常甜润的。但若是希望口感更SMOOTH，请陈年。

网友点评
张博钧烟斗丝品鉴WESSEX维吉尼亚金砖。好砖，绝对好砖，杠杠的好砖，极爱之，一块砖加入半斤芝麻油腌制，一点燃浓浓的芝麻油香气，满口芝麻味，绝对极香无比，这芝麻选自阿尔及利亚山区的野生芝麻压制，再加入顶级v草，极香无比，强力推荐，人手一砖，你要问为啥加芝麻油，因为压榨完的芝麻渣要用来做烟叶的肥料。嗷嗷的顶级农家肥，这下子的烟叶绝对香，以后就叫绝香芝麻烟斗丝。张博钧烟斗丝品鉴WESSEX维吉尼亚金砖。
金砖的优势就是纯粹，所以天然的就带着一股潜力无穷的感觉。这个烟抽起来很舒服 新草的话纯粹、力度有余但变化不足，还带着丝丝不干净的杂气 不够醇厚。据说八年以上的金砖 很是值得期待。我想有一定道理 尽管没试过 但是这种砖头的草普遍还挺值得放的 狐狸 折刀等等……
味道非常纯正！就是价钱嫌高了，替换豪海当主粮的话得加倍增加开支了，还是当“加菜”来享用比较合适。
不容易点燃，开封后一直湿湿地，根本就不灵
很多人都认为金砖很好，可我认为这就是款相当普通的v。在我的理解里，好的v必须醇厚但不霸道，甜味足但不腻歪，如果再可以层次丰富久抽不烦则更属上品。但金砖真的很抱歉，甜味不足，毫无层次，只有厚度尚属中等。这样的v草我不敢说比比皆是，但绝对没有好的让人想大量囤货。总看到有人几十块的囤，真是不能理解，莫非中国人特爱黄金所以爱屋及乌？？？这草买一块抽过就算了吧。在我的草库里（至少20款以上v草），wessex的这款金砖最多只能排倒数第三。
很特别的砖块包装，隔着包装摸起来感觉很干，但是打开后发现还是比较潮湿的，可以比较轻易地剥下一片，撕扯揉散成小指甲面积大小的片状，醒上20分钟，装斗(我用了pot的斗型)，比较容易点燃。这个是我至今抽过最耐烧的V草，非常非常耐烧！一斗抽了将近2个半小时，过程中点了3-4次火。口感上没有FVF那么甜，非常柔顺，适合星期天早上点上一斗，慢慢享受慵懒的周末，烟气不是那么复杂，但是非常舒服，像你在高中认识的恋人般羞涩，推荐！
和SG FVF……瑜亮之間….這是瑜
很明显的V草栗香，甜度较大。长期存放可抽真空，千万别手贱弄碎“陈化”，爱的就是他极度压缩的原滋原味
个人觉得比较耐抽，入口丝滑，无论大火小火口腔不会产生不舒服的感觉，甜甜的，得草如此，夫复何求，呵呵，有点夸张，不过确实值得一试，个人觉得比SG的FVF还要醇，压砖的中间部分特别香，也许和普洱紧压能保持茶香是一个道理吧？但就是上子弹麻烦，又要剪，又要磨，也因为好抽又变得不耐抽了，一砖很快就没了，哎，世事难两全啊。
值得一试，感觉还不错
