昨天入手一盒，切成3MM切片揉碎，打开醒烟一夜，第二天清晨一斗，开始几口颠覆了我对纯V的感觉，纯V还可以是这种味道，很像云南水烟筒的那种烟丝的味道，大失所望，但是随着进入中段，慢慢开始发生神奇的变化，这个草适合对于我来说，适合偶尔调剂，除非发烧级别，SG这个品牌，个人还是喜欢长期品尝小白和FVF~~补充一句，斗温真的很低··
绳烟，色泽油润漆黑，卷制精良，点燃之前闻起来像过去尝过的sg的一款口嚼烟，稍醒上几分钟，多了点类似薄荷的香味。初次点燃黑xx，感觉油味明显，像汽车机油又像炒菜油烟，完全不像过去尝试的其它v，只想将它封存到天荒地老。可是没有到10小时便又回像起那个味道，重点一斗慢抽细品，油味变成深度发酵的熟香，让人欲罢不能，斗温极低，享受了近一小时，站起身才感觉有些晃荡，sg家的草第一次有晕烟感，才想起斗友们对它劲道的评价。现在黑xx是常备草，收入250g装4包，每天必搞一斗。给玩斗朋友们都试了试，没一个第一斗抽完的，之后也只有一位朋友回味后想进货，黑xx可算是草中榴莲，和爱尔兰切一样新玩家勿碰
一个字“爽”
很多菸友對此物避之唯恐不及，偏偏俺就特愛此物。用剪子將其剪成二、三片2mm半圓片，再用手將菸葉仔細分開、裝入大斗，點火後慢抽，濃厚的雪茄香撲鼻而來，十分刺激，十分享受。用鼻腔細品，煙氣沁涼、醇厚、冗香、迷醺，值得讓人細細品嘗。
此物只应天上有，人间哪得几回闻！绝对的好东西，看似狗屎，但是做工精良，偌大的烟叶卷成的烟绳，没有一点烟梗，外表乌黑油亮，内里层次分明，试问谁家的宠物狗能拉出此等极品之狗屎？用小刀切3毫米宽的薄片，揉松后分三层填入斗钵，做燃烧层用了3根火柴，压实后再次点火，烟味很醇厚，略带些许苦味，但是稍纵即逝，随之而来的是苦尽甘来，中段部分空气中开始弥漫雪茄的气味，非常高雅，没有任何香精调味，纯烟丝能做出此等效果，让我辈饮水思源，创造此物者，真乃大师也，此物真乃是烟丝界之传奇。抽完一斗，从朋友家驱车回家，注意力颇不集中，有一种喝了黄酒后劲上来的感觉，浑浑噩噩中，不知驱车往何处，又是如何到了家门口，若是中途被交警拦下测酒精，是否会不过关，看来此物有一种能让人暂时抛却压力，忘却烦恼之功效，大麻乎？冰毒乎？太雕乎？鸦片乎？无论什么乎，每日呼上一呼，逍遥自在忘却世间一切纷扰，何乐而不为乎？
打开盒子后，发现一个U型长条，真的很像狗屎。切成3MM左右的小片，细细揉碎后装斗。刚开始几口，简直有一种迎头痛击的感觉，一度然我无法继续下去。坚持到1/3以后，感觉出来了，凉、甜、香、醇厚、变化丰富；有烤坚果的香味、芝麻糊的香味。抽到后面，头皮发麻，手开始颤抖，醉烟不可避免。最好不要用3MM烟道的斗，烟气太浓，而9MM有似乎有点不过瘾，6MM对我来说刚刚好；总之是一款很不容易降服纯V。建议小斗尝试，大斗必醉。
这款草可又是一杀人利器啊，绝对够猛。新手请勿尝试。这款调配变化丰富，非常浓郁，劲道超强。我抽这款草其实很矛盾，我被它浓郁的口感和丰富的变化所吸引，可又怕它那超强的劲道。每当我深深的抽一口，就犹如它对我迎头痛击一次。可以说我对它是又爱又怕

哪里能买到此草
很多草都是看了排名后才有想法尝试看到底好到哪里。FVF 四星里第一，黒木四星里第二。这两款草同时全V草，不含百里可，白肋，同时都是深度发酵。去掉了各自原有的辛辣使主体醇和，发酵后在不搀和调味草的情况下，产生了变化提高了层次感。试过并抽了一段时间后感觉。好的V草的都是做把烟草本身醇厚还原发挥到极致。 FVF风格像发酵版的顶级烤烟（烤过后的棕V为主），黒木就是发酵版得顶级混合烟（黒木里主要是黑V与红V的搭配） 。 黒木比FVF要劲小些。 但风味和层次要相对FVF多。开罐后闻到的是植物发酵后的酸，很开胃很吸引人。点火前段烟草被发酵后其本身天然的香味占据。中段湿度较低烟草主体的醇和与柔美发挥了出来。即便是在屋里也可以感觉到阳光下的田园间的宁静安逸。对抽贯浓烈的奥德赛，达芬奇，和帕塔加斯罗步图的我无比的享受。FVF和黒木不用谁高谁低。一个浓醇，一个柔美。斗客各有钟爱。国内市场供应和FVF情况差不多。寻百家只为觅钟爱。国外市场好一些。价格要论斤来购买加上运费和消费税折和差不多每50克60元左右。国内市场80有点高啊
未开罐摇起来沙沙响，以为干掉了，其实是散切片，且湿度很好。 开罐一股番茄味。似乎MCC的招牌就是一股蔬菜、水果的酸，很开胃。前段较凶，V的甘醇、浓烈扑面而来，但仅仅是一闪而过。中段那股狠劲渐渐退尽，换来的是平顺、柔美的甘甜，酸气也较之明显。温度非常理想，不咬舌，不呛喉。适合一直叼斗着做事，不会给你波澜不惊的感觉。抽起来很放松。尾端变化不大。总觉的MCC的草特别适合当零食，而且是很不错的零食小品。

今天拿到了一盒2013年2月的BF U/S切片，不知道是不是美国佬陈化的好，打开的时候已经析出了一层亮晶晶的糖霜，闻上去的味道很类似SG的BBF，一股很典型的英国V草香味。这款草V草含量据说为83.5%，B草有12.5%，剩下4%是Dark Fired Leaf。抽上去前面几口有股湖区味，很典型的GH味道，尽管GH声称这草没有添加这味道，但事实上还是有。好消息是只有几口有这味道，之后就感觉不大出来了。B草味道很明显，有坚果味，再加上中等的甜味，毫无咬舌感。一斗到底，非常舒服。不明白为什么这么久就只有2个点评。我认为非常好的一款草，GH果然是用料相当厚道。好草，很庆幸一次就买了500G的大包装，可以慢慢享受。

强劲、厚实、口感极浓郁、抽着很过瘾，个人认为比布朗2#更具代表性，湖区味就应该是这样的，口感调清淡了反而失去了湖区味的精髓
