尝百草阶段基本人手一罐……很湿的草，点燃后觉得食之无味弃之可惜，只能理解这个草适合新手练习点烟斗和练习如何抽……谈不上享受，谈不上层次……比较失望的一款，曾经总觉得是主观臆断，就装了一斗马坝原味，一斗拉森……一比较差距就出来了……
垃圾草 不多说什么了
这款还不错
现入门正在品尝此草，劲道很小，偶尔出甜味，烟气不烦人，很香。此草很难一斗到底，感觉留下很多残渣，不解。
感觉：不爽，香精够多！体会：适合刚刚上手的新斗客。接触了纯V和L草的，还是离他远点吧！好心老哥哥送了一大罐子，足足150G，我该咋消化呢？
我很奇怪这破玩意也号称适合新手么，我抽着味道很不咋地而且很难点着。亏了我作为新手一上来抽的是拉森权杖和法官切片。。。要真是先抽了他那说不准以后也不再抽斗了。
作为新手还不错的选择，反正我用这个替换了船长了
