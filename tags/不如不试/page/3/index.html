哪里能买到此草
很多草都是看了排名后才有想法尝试看到底好到哪里。FVF 四星里第一，黒木四星里第二。这两款草同时全V草，不含百里可，白肋，同时都是深度发酵。去掉了各自原有的辛辣使主体醇和，发酵后在不搀和调味草的情况下，产生了变化提高了层次感。试过并抽了一段时间后感觉。好的V草的都是做把烟草本身醇厚还原发挥到极致。 FVF风格像发酵版的顶级烤烟（烤过后的棕V为主），黒木就是发酵版得顶级混合烟（黒木里主要是黑V与红V的搭配） 。 黒木比FVF要劲小些。 但风味和层次要相对FVF多。开罐后闻到的是植物发酵后的酸，很开胃很吸引人。点火前段烟草被发酵后其本身天然的香味占据。中段湿度较低烟草主体的醇和与柔美发挥了出来。即便是在屋里也可以感觉到阳光下的田园间的宁静安逸。对抽贯浓烈的奥德赛，达芬奇，和帕塔加斯罗步图的我无比的享受。FVF和黒木不用谁高谁低。一个浓醇，一个柔美。斗客各有钟爱。国内市场供应和FVF情况差不多。寻百家只为觅钟爱。国外市场好一些。价格要论斤来购买加上运费和消费税折和差不多每50克60元左右。国内市场80有点高啊
未开罐摇起来沙沙响，以为干掉了，其实是散切片，且湿度很好。 开罐一股番茄味。似乎MCC的招牌就是一股蔬菜、水果的酸，很开胃。前段较凶，V的甘醇、浓烈扑面而来，但仅仅是一闪而过。中段那股狠劲渐渐退尽，换来的是平顺、柔美的甘甜，酸气也较之明显。温度非常理想，不咬舌，不呛喉。适合一直叼斗着做事，不会给你波澜不惊的感觉。抽起来很放松。尾端变化不大。总觉的MCC的草特别适合当零食，而且是很不错的零食小品。
